              # 读阮一峰周刊有感，谢谢你让我看见世界

              # 前言

              初识阮一峰，源于一篇微信订阅号的推送 🔔，我都不记得什么时候加的关注，但网络世界就是这么神奇👏。今天我想说一说我对他的认识以及他对我的影响

              # 无数个为什么？

              我一直好奇🤔，如此多的内容输出，他哪儿来的那么多时间，又从哪儿收集到了这些有趣又好玩的东西，又是怎样把这些碎片化的东西交织🖇在一起的，他精简又深奥的写作风格是与生俱来的吗💯，又是怎样做到Github Users Global Ranking 全球第二

              答

              勤奋+独立思想

              大家大概都知道，阮一峰有一个周刊专栏，会将他一周看到的、听到的好玩的信息分享出来，有国内的也有国外的，有科技的也有生活的...，这些东西再加上作者自己的理解呈现出来，比如：一些新型的软件已经将一些小型软件或者传统软件代替、全国每年的垃圾邮件将会造成多少的能源损失、以及互联网的饱和...，这些问题像我一样的普通人或许从来都没有思考过，但他确以图文的形式展现出来了。他将自己的变成大家的，将遥远的变成触手可及的。

              # 为何偏爱

              他给我指引了一条快速入门的道路，在这个道路上我看到了希望。

              就拿我最近学习docker来说吧：首先我去浏览了官网，一堆密密麻麻的英文，然后（what，why，where)涌上心头，这时候就开始各种搜索，会有一系列的小文章出现，为什么说是小文章，因为这些文章千篇一律，大体流程就是教你怎么安装、说下docker是什么以及docker的优势，巴拉巴拉就没了，是的，就没了，此时我的状态又是（how，when,which)，心好累啊！

              直到搜到阮的《docker入门教程》，我就知道真的离入门不远了 👋 ...

              技术博客那么多，为什么我独爱他的文章，我大概总结了有那么几点：

              • 他的文章，有始有终、有举例
              • 手把手教学，然后跟着step by step就上手了，非常适合初学者
              • 完美的注释
              • 主题明确，他会围绕主题展开简单的、直白的教程，没有掺杂其他复杂的东西

              在这之后，就有了基本的认知，然后就开始有目的的查找了，久而久之，成功率 + 提升 == 信服，所以我会一直追随

              # 他的荆棘

              俗话说，树大招风，成功的路上并不是一帆风顺，如果你对他有所了解，你就会知道他网站被攻击的事情吧，当时他说：有人真的恨他，即使他没有伤害任何人的利益。是啊，正如他所说，他的所有博客免费观看，所有知识免费共享，确实没有伤害到谁的利益。就算博文中有广告信息，我倒觉得只不过是他用自己擅长的方式给家人一口饭吃。如果攻击者真的认为跟阮的观点有冲突，既然作者开通了评论权限，不妨试试给作者留言，来一场真正的对话，而不是犯如此低级又愚昧的错误

              忠告

              真正的成功绝不是打压被人，而是提高自己，希望此类事件不要在发生

              # 牛人代表作

              放眼中国，比较出名的前端大牛并不多，我熟知的大概有廖雪峰、张鑫旭、李炎恢、大漠...，他们都有一个共同的特点，就是对外有所输出，无论是以博客的形式还是以视频的形式

              牛人代表作

              张鑫旭的《css世界》、李炎恢的《javascript视频教程》、廖雪峰的《javascript教程》、阮一峰的《ECMAScript 6 入门》

              # 开启我的BLOG大门

              随着时间的推移，我发现我们在工作中的作品都渐渐的下线或者淘汰了，这些code再次被利用或者参考的价值非常低，甚至每到一个时期我都会怀疑怎么写了这么烂的code，所以我决定要把这些隐士改变收集起来，同时也把每个项目中的遇到的难点和解决问题的思路记录下来，让之前的经验以文字而不是以代码的形式存储起来，以便于未来做参考。

              所以从今天起就正式开始我的blog之旅，没有什么太大的愿景，只希望N年后当我离开这个行业的时候，我能笑着说我有输出😊
